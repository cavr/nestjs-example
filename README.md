## Description

Simple Api Rest Example with NestJS (NodeJS TypestScript Framework)

Swagger localhost:3000/api

https://github.com/nestjs/typescript-starter

https://codebrains.io/structured-expressjs-nestjs-swagger-mongodb-typescript/

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
